> **Trip Odyssey**는 그룹 여행에서의 **역할 분담**을 기반으로,  
> 여행 시작부터 마무리까지 각 단계에 필요한 기능을 제공하여  
> 보다 **쾌적하고 효율적인 여행 경험**을 지원하는 **AI 기반 여행 보조 서비스**입니다.

---

## 🧭 프로젝트 개요

> “여행 준비, 이제는 게임처럼!”

**Trip Odyssey**는 그룹 내 **역할 분담**을 중심으로,  
각 멤버의 역할에 꼭 맞는 기능을 제공하여  
여행 준비의 복잡함을 **쉽고 체계적으로 해결해주는 All-in-One 웹 플랫폼**입니다.  

교통/숙소 담당, 일정 담당, 재무 담당 등 역할별로 제공되는 기능을 통해  
여행 준비 과정이 **협업 중심의 게임처럼 재미있는 경험**이 되도록 설계되었습니다.  

또한, **AI 기능을 통해 일정 최적화, 지출 리포트 생성, 교통정보 등록 자동화** 등  
세부 작업을 보다 편리하게 도와줌으로써  
전체적인 사용자 경험을 한층 더 향상시킵니다.

## 💡 주요 기능

## 1. 그룹 생성 & 멤버 관리

|  | 기능명           | 기능 요약 설명                                                               |
|--|------------------|----------------------------------------------------------------------------|
| 1-1  | 멤버 초대 | 방장 전용 페이지에서 "멤버 초대" 버튼을 클릭하고, 아이디 검색 후 초대를 전송할 수 있음 |
| 1-2  | 그룹 설정        | 그룹명, 여행 기간 등 그룹 정보를 입력하여 그룹을 생성하고 관리할 수 있음            |


## 2. 일정 조율 

|  | 기능명           | 기능 요약 설명                                                               |
|--|------------------|----------------------------------------------------------------------------|
| 2-1  | 일정 조율 진입   | 그룹 메인에서 "일정 조율" 메뉴를 클릭해 캘린더 기반 일정 조율 화면에 진입         |
| 2-2  | 개인 일정 입력   | 날짜를 클릭하거나 드래그하여 자신의 가능한 일정을 손쉽게 입력할 수 있음           |
| 2-3  | 겹치는 날짜 확인 | 멤버별로 입력한 일정 중 겹치는 날짜가 자동으로 하이라이트되어 최적 날짜를 파악 가능  |
| 2-4  | 여행 상태 변경   | 여행 상태 드롭다운에서 "일정 조율 완료" 등 단계별로 상태를 변경할 수 있음         |


## 3. 장소 검색 & 추천

|  | 기능명            | 기능 요약 설명                                                               |
|--|-------------------|----------------------------------------------------------------------------|
| 3-1  | 지도 페이지 접속  | 그룹 내 "장소 찾기" 메뉴를 클릭해 전체 지도 기반 여행지 검색 화면에 진입                     |
| 3-2  | 여행지 검색       | 검색창에 장소명을 입력하면 지도에 해당 위치가 마커로 표시됨                    |
| 3-3  | 장소 추천         | 지도 마커 클릭 시 상세 정보를 확인하고, "추천하기" 버튼으로 팀원에게 추천 가능     |
| 3-4  | 추천 장소 좋아요  | 추천된 장소 목록에서 하트(좋아요) 버튼을 눌러 팀원 추천 장소에 반응할 수 있음      |


## 4. 일정 관리 

|  | 기능명            | 기능 요약 설명                                                               |
|--|-------------------|----------------------------------------------------------------------------|
| 4-1  | 일정 관리 진입    | 그룹 내 "일정 계획" 메뉴를 클릭해 추천 장소 리스트와 일정 관리 화면에 진입                |
| 4-2  | 일정 카드 만들기  | 추천 장소를 선택해 일정 카드로 추가할 수 있음                                 |
| 4-3  | 날짜별 일정 배치  | 일정 카드를 드래그&드롭으로 원하는 날짜에 배치할 수 있음                        |
| 4-4  | 일정 순서 조정    | 같은 날 내에서 일정 카드의 순서를 자유롭게 변경 가능                            |
| 4-5  | AI 일정 최적화    | 좌측 하단 캐릭터 버튼을 클릭하면, AI가 동선을 고려해 일정 순서를 자동으로 재배치함         |


## 5. 교통 & 숙소 관리 

|  | 기능명            | 기능 요약 설명                                                               |
|--|-------------------|----------------------------------------------------------------------------|
| 5-1  | 교통/숙소 진입    | 그룹 내 "교통/숙소" 메뉴를 클릭해 교통 및 숙소 관리 페이지에 진입                        |
| 5-2  | 교통편 카드 추가  | "교통편 추가" 버튼을 눌러 KTX 등 교통 정보를 카드 형태로 추가할 수 있음           |
| 5-3  | 항공편 정보 추가  | 항공편의 상세 정보를 입력해 여러 교통수단을 통합 관리할 수 있음                   |
| 5-4  | 숙소 정보 추가    | "숙소 추가" 버튼으로 호텔 등 숙소 정보를 체계적으로 입력 및 관리할 수 있음          |
| 5-5  | AI 티켓 인식      | 좌측 하단 캐릭터 버튼을 클릭하면, AI가 티켓 정보를 자동 추출해 카드로 생성함                   |
| 5-6  | 여행 상태 변경    | 상태 드롭다운에서 "계획 완료" 등으로 여행 진행 상태를 변경할 수 있음               |


## 6. 재무 관리 

|  | 기능명            | 기능 요약 설명                                                               |
|--|-------------------|----------------------------------------------------------------------------|
| 6-1  | 재무 페이지 진입  | 그룹 내 "재무" 메뉴를 클릭해 재무 관리 대시보드에 진입                               |
| 6-2  | 지출 내역 기록    | 각 그룹 일정 우측 + 버튼으로 금액, 카테고리별로 지출 내역을 실시간 입력할 수 있음          |
| 6-3  | 영수증 정보 추가  | 각 지출 항목에 대해 장소, 시간, 금액 등 상세 정보를 입력할 수 있음                 |
| 6-4  | AI 지출 분석      | 좌측 하단 캐릭터 버튼을 클릭 시 카테고리별 지출 비율, 일별 추이 등 시각적 분석 그래프 제공        |
| 6-5  | 개인별 입금 현황  | 멤버별 입금 상태를 체크리스트로 확인하고 실시간 추적할 수 있음                     |

## 🚀 실행 방법


### ✅ 프로젝트 클론
1. **git clone**
    ```bash
    git clone https://lab.ssafy.com/ssafy_13th_18class/999_final/ssafy_trip_final_kimjoonhyeok_parkjihun_bonghyenjin
    ```
2. **프로젝트 진입**
    ```bash
    cd ssafy_trip_final_kimjoonhyeok_parkjihun_bonghyenjin/
    ```

### ✅ 백엔드 실행 (Spring Boot)
1. **백엔드 디렉토리 진입**
    ```bash
    cd backend
    ```
2. **실행 권한 부여 및 실행**
    ```bash
    chmod +x mvnw
    ./mvnw spring-boot:run
    ```

### ✅ 프론트엔드 실행 (Vue)

1. **프론트엔드 디렉토리 진입**
    ```bash
    cd frontend
    ```
2. **의존성 설치 및 dev모드로 실행**
    ```bash
    npm install 
    npm run dev
    ```
3. **웹페이지 진입**
    ```
    http://localhost:5173/login
    ```
4. **회원가입/로그인 후 서비스 이용 가능**
